# Contributing

Thank you for contributing to FacturAutentico.

## Scope

This project handles fiscal workflows and mock/staged integrations.
Changes must prioritize correctness, traceability, and operational safety.

## Branching

- Base branch: `main`
- Feature branches: `feat/<scope>`, `fix/<scope>`, `chore/<scope>`

## Pull Request Rules

- Keep PRs focused and reviewable
- Explain business context and technical impact
- Include rollback notes for risky changes
- Update docs when behavior changes

## Commit Style

Prefer Conventional Commits:
- `feat:` `fix:` `docs:` `chore:`
